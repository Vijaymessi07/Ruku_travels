# Image Assets

I have structurally configured the website to look stunning right out of the box using high-quality Unsplash image placeholders that match the descriptions of the images you uploaded. 

To use your exact uploaded images, follow these steps:
1. Rename your logo file to `logo.png` (or `.jpg`) and place it here. Then update `index.html` line 34 to point to `assets/images/logo.png`.
2. Place your 4 travel images in this folder.
3. In `index.html`, search for `https://images.unsplash.com/` and replace those URLs with the relative paths for your images, for example: `assets/images/your-travel-image.jpg`.

*Folders:* 
- `assets/images/` (this folder)
- `css/` (styles)
- `js/` (scripts)
